## ⚙️ 核心设置

### 1️⃣ Sleep Mode（睡眠模式）
**作用**：总开关，控制是否启用音频处理

- ✅ **开启**：激活所有音频优化功能
- ❌ **关闭**：音频直接播放，不做处理

**建议**：睡前观看时开启

---

## 🎵 音频处理

### 2️⃣ Stability Level（稳定性级别）
**作用**：控制音量波动的程度

| 档位 | 效果 | 适合场景 |
|-----|------|---------|
| **Light** | 温和压缩，保留动态 | 音乐视频、高品质内容 |
| **Medium** ⭐ | 平衡的音量控制 | 大多数视频、日常观看 |
| **Strong** | 最大程度稳定音量 | 惊悚片、动作片、深夜 |

**例子**：
- Medium: 对话 🔊🔊 → 爆炸 🔊🔊🔊🔊 (差距 2倍)
- Strong: 对话 🔊🔊🔊 → 爆炸 🔊🔊🔊🔊 (差距很小)

---

### 3️⃣ Sound Character（声音特性）
**作用**：调整声音的柔和程度

| 档位 | 效果 | 适合场景 |
|-----|------|---------|
| **Natural** | 保持原声 | 高品质内容、喜欢原音 |
| **Gentle** ⭐ | 柔和舒适 | 大多数内容、长时间观看 |
| **Soft** | 极度柔和 | 深夜、噪音敏感、耳朵疲劳 |

**技术说明**：
- 削减刺耳的高音
- 减少轰鸣的低音
- 增强舒适的中频（人声）

---

## 🎤 语音增强

### 4️⃣ Voice Focus（语音聚焦）
**作用**：突出人声，降低背景音

- ✅ **开启**：人声更清晰，背景音减弱
- ❌ **关闭**：保持原始平衡

**适合**：访谈、播客、演讲、教育内容
**不适合**：音乐视频

---

### 5️⃣ Background Reduction（背景音削减）
**作用**：控制背景音的音量（需开启 Voice Focus）

```
0 dB   →   -12 dB
不削减      最大削减
```

| 级别 | 背景音保留 | 适合场景 |
|-----|-----------|---------|
| 0 dB | 100% | 轻度增强人声 |
| -3 dB | 70% | 有轻微背景音乐 |
| -6 dB | 50% | 正常访谈节目 |
| **-9 dB** ⭐ | 30% | 大多数播客 |
| -12 dB | 25% | 背景音很吵的内容 |

---

## 🔊 音量控制

### 6️⃣ Auto Gain Control（自动增益控制）
**作用**：让不同视频的音量保持一致

**效果对比**：
```
关闭 AGC:
视频A: 🔊🔊🔊🔊🔊🔊 (很响) ❌ 需要手动调
视频B: 🔊🔊         (很轻)

开启 AGC:
视频A: 🔊🔊🔊🔊 (自动降低) ✅ 自动统一
视频B: 🔊🔊🔊🔊 (自动提升)
```

**建议**：开启，特别是播放列表自动播放时

---

### 7️⃣ Target Loudness（目标响度）
**作用**：设定统一的音量目标（需开启 AGC）

```
-24 LUFS  →  -18 LUFS  →  -12 LUFS
  极轻         适中         很响
```

| 级别 | 音量 | 适合场景 |
|-----|------|---------|
| -24 LUFS | ■□□□□ 20% | 深夜、浅睡眠 |
| -21 LUFS | ■■□□□ 40% | 夜间安静环境 |
| **-18 LUFS** ⭐ | ■■■□□ 60% | 日常观看（YouTube标准） |
| -15 LUFS | ■■■■□ 80% | 白天、嘈杂环境 |
| -12 LUFS | ■■■■■ 100% | 嘈杂环境（电视标准） |

---

### 8️⃣ Output Volume（输出音量）
**作用**：整体音量的微调

```
-12 dB  ←  0 dB  →  +12 dB
 降低      不变      提升
```

**使用建议**：
- **-3 to -6 dB**：深夜、耳机太响
- **0 dB** ⭐：正常情况
- **+3 to +6 dB**：视频音量太小
- ⚠️ **>+6 dB**：可能失真---

## 🎯 快速配置方案

### 🌙 深夜睡觉
```
Sleep Mode:     ✅ ON
Stability:      Strong
Sound:          Soft
Voice Focus:    ❌ OFF
AGC:           ✅ ON
Loudness:      -21 LUFS
Output:        -3 dB
```
**效果**：音量极稳、声音柔和、不会被吵醒

---

## ❓ 常见问题速查

| 问题 | 解决方案 |
|-----|---------|
| 音量还是会突然变化 | Stability → Strong |
| 声音闷闷的 | Sound → Natural 或 Gentle |
| 人声不清晰 | Voice Focus → ON，Background → -9dB |
| 不同视频音量差很大 | AGC → ON |
| 音质变差 | Stability → Light，Sound → Natural |
| 处理后太小声 | Loudness → -15，Output → +3dB |

## 🔍 名词解释

- **LUFS**：响度单位，符合人耳感知
- **dB**：分贝，音量单位
- **压缩**：缩小音量波动范围
- **AGC**：自动增益控制
- **动态范围**：最大音量和最小音量的差距
