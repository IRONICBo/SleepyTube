## 🎯 四种场景模式

### 1. ⭕ OFF（关闭）

**适用场景**：
- 🎵 高品质音乐欣赏
- 📺 已经音质很好的视频
- 🎮 需要原汁原味的游戏直播

**效果**：
```
✅ 完全关闭音频处理
✅ 保留100%原声
✅ 零延迟、零失真
```

**配置**：
```yaml
Sleep Mode: ❌ OFF
所有处理: 关闭
```

---

### 2. 🌙 Sleep（睡眠模式）

**适用场景**：
- 💤 睡前观看视频
- 🌃 深夜放松
- 😰 惊悚/恐怖片
- 💥 动作片（防止爆炸声）

**效果**：
```
✅ 最大程度音量稳定
✅ 超柔和音质，不刺耳
✅ 音量偏轻，适合睡眠
✅ 绝不会被突然的声音吵醒
```

**配置**：
```yaml
Stability Level: Strong     (最大稳定性)
Sound Character: Soft       (超柔和)
Voice Focus: OFF
Auto Gain Control: ON
Target Loudness: -21 LUFS   (音量较轻)
Output Volume: -3 dB        (再降低一点)
```

**原理**：
- 强力压缩比 (12:1) 让音量波动最小
- 削减刺耳的高频和轰鸣的低频
- 降低整体音量到舒适水平

---

### 3. 🎙️ Podcast（播客模式）

**适用场景**：
- 🎙️ 播客、访谈节目
- 📰 新闻播报
- 📚 教育讲座
- 🎓 在线课程
- 🗣️ 演讲视频

**效果**：
```
✅ 人声极度清晰
✅ 背景音乐不干扰
✅ 音量适中舒适
✅ 长时间听也不累
```

**配置**：
```yaml
Stability Level: Medium     (平衡压缩)
Sound Character: Gentle     (柔和)
Voice Focus: ON             (★关键特性)
Background Reduction: -9 dB (背景音降低70%)
Auto Gain Control: ON
Target Loudness: -18 LUFS   (标准音量)
Output Volume: 0 dB
Speech Rate: ON             (★自动语速调整)
Target Rate: Normal         (3.5 音节/秒)
```

**原理**：
- Voice Focus 增强人声频段 (300Hz-3400Hz)
- 背景音削减让对话更清晰
- 中度压缩保持自然感
- 语速检测自动调整播放速度，让快速说话更易理解

---

### 4. 🎬 Movie（电影模式）

**适用场景**：
- 🎬 观看电影
- 📺 电视剧、网剧
- 🎮 游戏直播/录播
- 🎪 综艺节目
- 📹 Vlog视频

**效果**：
```
✅ 防止突然巨响
✅ 保持一定动态
✅ 声音较柔和
✅ 音量适中
```

**配置**：
```yaml
Stability Level: Strong     (强稳定性)
Sound Character: Gentle     (柔和)
Voice Focus: OFF            (保持平衡)
Auto Gain Control: ON
Target Loudness: -18 LUFS   (标准音量)
Output Volume: 0 dB
```

**原理**：
- 强压缩防止爆炸声、尖叫声
- 不开启 Voice Focus，保持音乐和音效平衡
- 柔和的音质让长时间观看更舒适

## ⚙️ 高级设置

点击右上角的 **齿轮图标** 可以打开高级设置面板，进行细粒度调整。

### 何时使用高级设置？

1. **微调预设场景**
   - 场景不完全符合需求
   - 想要更个性化的配置

2. **创建自定义配置**
   - 当你调整任何参数时
   - 自动切换到 "Custom" 场景
   - 设置会被保存

3. **深度定制**
   - 了解每个参数的作用
   - 针对特定内容优化
包含：

#### 音频处理
- **Stability Level**: Light / Medium / Strong
- **Sound Character**: Natural / Gentle / Soft

#### 语音增强
- **Voice Focus**: 开/关
- **Background Reduction**: 0 ~ -12 dB

#### 音量控制
- **Auto Gain Control**: 开/关
- **Target Loudness**: -24 ~ -12 LUFS
- **Output Volume**: -12 ~ +12 dB

## ❓ 常见问题

### Q: 我能修改预设场景吗？
**A**: 可以！点击高级设置调整任何参数。调整后会自动切换到 "Custom" 场景并保存你的设置。

### Q: 场景之间可以快速切换吗？
**A**: 可以！直接点击不同场景，配置立即生效，无需确认或保存。

### Q: Sleep 模式音量太小了？
**A**: 打开高级设置，调整 Target Loudness 到 -18 LUFS 或 Output Volume 到 0 dB。

### Q: Podcast 模式背景音还是太大？
**A**: 打开高级设置，将 Background Reduction 调到 -12 dB（最大值）。

### Q: Movie 模式还是会被吓到？
**A**: 尝试 Sleep 模式，它有最强的压缩效果。

### Q: 音质变差了怎么办？
**A**: 切换到 OFF 模式，或在高级设置中将 Sound Character 改为 Natural。
